# Freelance Performance Dashboard

A modern and responsive freelance performance dashboard built with **React**, **TypeScript**, and **Vite**.  
The project is designed to help freelancers monitor productivity, organize workflow, and present performance data in a clean and professional interface.

## Live Demo

[View Live Demo](https://hassanjaberdev.github.io/Freelance-Performance-Dashboard/)

## Features

- Clean and modern dashboard UI
- Responsive design for different screen sizes
- Built with reusable React components
- Organized project structure
- Fast development and build performance using Vite
- Type-safe codebase with TypeScript

## Tech Stack

- React
- TypeScript
- Vite
- CSS
